## 基础组件

### 安装

::: code-tabs#shell
@tab:active yarn
```bash
yarn add @brewer/anthill-core
```

@tab npm
```bash
npm i @brewer/anthill-core
```

:::

### 使用

```typescript
import { BaseSearchBar } from '@brewer/anthill-core';
```

## 业务组件

```sh
yarn add @brewer/anthill-core-biz
```

### 使用

```typescript
import { BaseLogin } from '@brewer/anthill-core-biz';
```
